Features

Hunter and Speedrunner Roles

Easily assign roles to players. Hunters chase and Speedrunners evade, creating an exciting gameplay dynamic.

Game Management

Start, stop, and manage the game with simple commands. Keep track of the game status effortlessly.

Wide Compatibility

Works on Minecraft versions from 1.12.2 to the latest, ensuring broad compatibility across servers.

Intuitive Commands

Simple and intuitive commands to control the game, making it easy for both players and admins.

Permission System

Robust permission system to manage who can use and administer the game.

Open Source

Fully open-source project. Contribute, modify, and improve the plugin as you see fit.

Commands and Permissions

Commands

ManhuntPlus provides a set of commands to manage the game:

Command Description
/manhunt hunter <player> Add a player as a hunter.
/manhunt speedrunner <player> Add a player as a speedrunner.
/manhunt start Start the Manhunt game.
/manhunt stop Stop the Manhunt game.
/manhunt status Show the current game status.
/manhunt reload Reload the plugin configuration.

Permissions

Permissions allow you to control access to various commands:

Permission Node Description
manhunt.use Allows access to basic Manhunt commands.
manhunt.admin Allows access to all Manhunt commands.
manhunt.bypass Bypasses Manhunt game restrictions.

Dependencies

ManhuntPlus supports and requires certain libraries for enhanced functionality:

  • ProtocolLib: Required for certain advanced features of the plugin.
  • PlaceholderAPI: Supported for integration with placeholders for dynamic content.
